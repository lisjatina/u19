Implement chat server

  1. Investigate jtm.extra13 package
  2. Implement class ChatServer to pass unit tests

  3. Server could be checked using nc program, e.g. following:

    nc 127.0.1.1 7701

  4. To check if server is listening on necessary port, run command:

    netstat -antp|grep 7701

  5. If necessary, server can be killed with command:

    kill -9 pid
    wher pid is process id from the previous command
